Resources

Now it is going to mention the resources made by others and used to develop this software, it shall be considered that this resources are important to configure the software but they are not the finality on it.

  • AntiWord: It is a software developed by Adri van Os (http://www.winfield.demon.nl/) capable of transform Microsoft Word files into plain text. The use of this software let the user has the possibility of use Anticopia software to analyze Word files.
  • JEditorPane images: Java functions don't let to include in a JEditorPane images with a relative contain in a html. Rob Kenworthy from JavaWord.com wrote an article (http://www.javaworld.com/javatips/jw-javatip109.html) that describe the problem and arrange the possibility of download several Java classes developed by him and let fix that problem. The used Java classes have been MyHTMLEditorKit.java and MyImageView.java. Considering that it is a complete and fixing develop inside any Java software, the author hasn't wanted to modify, just use the resource develop by Rob Kenworthy and be thankful because of his software.
  • Images: All the images which appear in the software have been got from the Web page http://www.iconfinder.net, it uses for its icons Creative Commons, GPL and LGPL licences.

Thanks to the people that have contributed with these resources, the developed of this software has been centred in the scope of it, this resources have done to the software an extra function and extra visual that the users of the AntiCopia software will thanks.
