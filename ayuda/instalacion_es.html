Instalación

En este primera versión de la aplicación no es necesaria ninguna instalación típica. Tan sólo se necesita descargar el software y localizarlo en cualquier carpeta del sistema respetando la estructura de ficheros y carpetas descargadas. Para el funcionamiento de la aplicación se necesita tener instalada la versión 6 de la máquina virtual Java (http://www.java.com/es/download/).

La aplicación, en esta primera versión, permite analizar contenido plano de cualquier documento, copiando del origen y pegándolo en el panel de texto, y documentos de Microsoft Office Word, hasta la versión 2003, apoyándose en el software AntiWord de Adri van Os que después se explicará. Para la correcta integración de Plagium Alert con AntiWord hay que descargar el software desde la web del autor (http://www.winfield.demon.nl/. Para sistemas Linux, como Ubuntu, puede bastar con instalarlo desde una consola (si se tienen establecidos correctamente los repositorios) ejecutando "apt-get install antiword". Para sistemas Microsoft Windows, tras descargar desde la web el archivo comprimido, hay que extraer el contenido en "C:\AntiWord". Es importante establecerlo exactamente en esa ruta.
