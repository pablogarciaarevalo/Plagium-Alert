Text analysis

Plagium Alert software analyze the contain of the text situated in the central panel of the screen. That's why, if the user require it, any contain will be analyzed. The software will analyze the contain stated in this central panel, no matter if the document is written, copied or pasted on it.

To make an analysis, execute Plagium Alert software and select "New" from the direct access button or from the menu File/New. After establish the wished text in the central panel, select Analyze from the direct access button or from the the menu File/Analyze. The time that Plagium Alert will take to analyze the document will depend on the computer CPU and RAM where the file will be executed, the Internet connexion speed, the established type of analysis (tiny speed by defect) and the limit of searches.

The analysis is continuously asking for searches to Google as if it were a drowse, analysing all the results from the WEB server and after studying the results to supply the user a understanding result.

The searches are realized in group of 20 to 20, so the software will join 20 words and search them in Google using quotation marks. This action let the software look for Internet and show the links that contain the exactly same 20 words in the same order.
