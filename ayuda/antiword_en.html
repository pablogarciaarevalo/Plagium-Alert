AntiWord

Antiword software is free software developed by Adri van Os and it can be found in his WEB site http:\\www.winfield.demon.nl\. It is available in the next operative systems: FreeBSD, BeOS, OS/2, Mac OS X, Amiga, VMS, NetWare, Plan9, EPOC, Zaurus PDA, MorphOS, Tru64/OSF, Minix, Solaris y DOS.

AntiWord convert the binary files from Word 2, 6, 7, 97, 2000, 2002 y 2003 to plain text and a PostScript.

Considering that most final users percentage work with Microsoft Office, the analysis of the contain of this type of documents make it prioritary. That is why, use this free software developed by Adri van Os has been considered, AntiWord software is quite stable but it is not the object of Plagium Alert software.
