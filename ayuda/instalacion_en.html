Install

In this first version is not necessary any installation. It is necessary download the software and place it in any folder just taking care about the structure of the downloaded files and folders. To make it works, it is necessary to install the version 6 of the Java Virtual Machine (http://www.java.com/es/download/).

This application, first version, let the user analyze the plain contain of any document, copying the original text and pasting it in the text panel; and Microsoft Office Word documents, till version 2003, using AntiWord software by Adri van Os that will be explain later. To get the correct integration between Plagium Alert software and AntiWord software, the user shall download AntiWord from the author web (http://www.winfield.demon.nl/). For Linux systems as Ubuntu, the user can install it from a console (if the repositories are correctly established) executing "apt-get install AntiWord". For Microsoft Windows system, after download the zip file from the web, the contain shall be unzip in "C:\AntiWord". it is important to place it in this path.
