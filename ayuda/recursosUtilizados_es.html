Recursos utilizados

A continuación se hace mención de los recursos utilizados de terceros que se han integrado con el núcleo de esta aplicación, así como la utilización dentro del programa observando que son partes imprescindibles del total de la aplicación pero no son objetivo del desarrollo del mismo.

  • AntiWord: Software desarrollado por Adri van Os (http://www.winfield.demon.nl/) capaz de transformar ficheros de Microsft Word en texto plano. La utilización de este elemento se realiza para dar al usuario un elemento añadido al utilizar el programa y poder utilizarlo directamente con ficheros Word.
  • Imágenes en JEditorPane: Las funciones de Java no permiten incluir en un JEditorPane imágenes con contenido relativo dentro de un html. Rob Kenworthy de JavaWorld.com escribió un artículo (http://www.javaworld.com/javatips/jw-javatip109.html) describiendo el problema y puso a disposición para la descarga de varias clases de Java que desarrolló que permite tal efecto. Las clases utilizadas han sido MyHTMLEditorKit.java y MyImageView.java. Puesto que es un desarrollo íntegro y acomodable dentro de cualquier aplicación Java, no se ha querido modificar sino tan sólo utilizar el recurso ya desarrollado por Rob Kenworthy y agradecerle su aportación.
  • Imágenes: Todas las imágenes que aparecen en la aplicación han sido conseguidas desde la página http://www.iconfinder.net que utiliza para sus iconos licencias Creative Commons, GPL y LGPL.

Gracias a las personas que han aportado estos recursos, el desarrollo de este software se ha centrado en el objetivo del mismo, pudiendo dar a la aplicación un añadido extra funcional y visual que los usuarios agradecerán.
