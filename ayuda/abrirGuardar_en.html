Open & save results

After finalize the analysis of the selected contain, Plagium Alert software let you save the result. On this way, you will have the possibility of analyze it and lately open it and check it without make another analysis. The results, that have been saved, will be stored in a .pga extension document.

When the document is saved, the software let you establish, as an option, the characteristics of the file (about the analyzed contain) as name, surname and identifier number of the author, title, subject, month and year; this is the way to personalize the obtained results and get a higher control about the realized analysis. These fields are defined as an option. If these fields are filled in, they will appear in the top of the "Advanced results".

At any moment of the program execution, any document, previously saved, can be opened.
