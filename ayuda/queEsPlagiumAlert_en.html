What is Plagium Alert?

Plagium Alert is a searches software and contain analyzer to automotive as much as it could, the user work to know if the contain of any text is original or a copy. To get it, Plagium Alert software use Google server as a searcher that look for document contain parts into Google.

Plagium Alert v1.0 es una Java software and multiplatform, so you can use almost in every operating system.

This is a software developed as a Final Career Project for the Escuela Universitaria de Informatica of the Universidad Politecnica in Madrid, and it is the first version to develop. Considering the TFC's of the rest of students, more versions will be provided adding new capabilities and improving the actual version.
