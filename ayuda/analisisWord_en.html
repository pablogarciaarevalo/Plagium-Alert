Microsoft Office Word analysis

Most of the documents are based on this type of text editor, Plagium Alert software include the option of analyze a Word document without copy and paste it in the principal text panel.

To get it, after open Plagium Alert software, select "New" and click the button "Browser". This button will open a window where select the required Word file. After select it and using the AntiWord software, this software will transform the text and the document will be overwritten in the principal text panel by Plagium Alert software. Realize that over this text the user could modify and analyze (after select the button "Analyze") the text which appear in the principal text panel.
