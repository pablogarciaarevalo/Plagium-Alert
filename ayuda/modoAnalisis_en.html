Analysis mode

From the menu Configuration/Configure check, the software will give the possibility to change the analysis type.

Configuration check let the user establish the following options:

  • Tyne search: The software search in to Google using the first 20 words (if it had it) of each paragraph.
  • Normal search: The software search in to Google from 20 to 20 words.
  • Exhaustive search: The software search in to Google using 20 words from 5 to 5 words.
Logically, as many searches in each paragraph would be done, the probability to make a correct analysis is higher. Anyway, It is not advisable make an exhaustive search, in case of it was a small paragraph, because this action can produce the stop of the service to the public IP direction from Google servers. Google servers can understand this action as a hacking attack. The consequences of modify this options is user responsibility.

The searches limit assure the availability of the system (to avoid Google servers cut the service) establishing just one search for the establish seconds number. Of course, this modification in the seconds number will cause that the analyze of the document takes long that if the software does not consider it, but the calc speed is not the object of the first version of the software.
